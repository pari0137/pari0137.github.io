5 HTML Tag Descriptions

anchor Tag
The a tag or anchor tag, creates a link to take the user somewhere else.
href Tag
The href tag tells the browser where the link should go.
nav Tag
This is the navigation of the page. This will contain the links to other pages of the website.
article Tag
The article tag is used for containing independent peices of content.
blockquote Tag
This is used for a large stand alone quote from another source.
